[2026-02-07 13:49] Here are some classic command-line fun tools available via Homebrew:Text & ASCII art:
- cowsay — ASCII cow says your message (cowsay "hello")
  - figlet — large ASCII text banners (figlet "hello")
  - toilet — colored ASCII text art
  - lolcat — rainbow colored output (pipe anything to it)

  Animations & visual:
  - sl — steam locomotive when you mistype ls
  - cmatrix — Matrix-style falling text
  - asciiquarium — animated aquarium in terminal
  - pipes-sh — animated pipes screensaver
  - nyancat — Nyan Cat animation

  Fun commands:
  - yes — prints a string forever (built-in)
  - rev — reverses text (built-in)
  - say "hello" — text-to-speech (macOS built-in)

  Games:
  - nethack — classic roguelike dungeon game
  - ninvaders — space invaders
  - moon-buggy — drive a buggy on the moon
  - greed — terminal puzzle game

  Install any with brew install <name>.
